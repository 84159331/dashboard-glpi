# Configuração do GitHub

## Opção 1: Criar novo repositório no GitHub

1. Acesse: https://github.com/new
2. Nome do repositório: `dashboard-glpi` (ou outro de sua preferência)
3. Escolha Público ou Privado
4. **NÃO** marque "Add a README file"
5. Clique em "Create repository"
6. Copie a URL do repositório (exemplo: https://github.com/SEU_USUARIO/dashboard-glpi.git)

## Opção 2: Usar repositório existente

Se você já tem um repositório, copie a URL dele.

## Depois de ter a URL, execute:

```bash
git remote add origin SUA_URL_AQUI
git branch -M main
git push -u origin main
```
